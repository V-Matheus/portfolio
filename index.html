Habilidades

Minhas Skills

HTML

HTML é uma linguagem de marcação para estruturar e apresentar conteúdo de páginas web com definição de hierarquia e formatação dos elementos.

CSS

CSS é uma linguagem de estilo que controla a apresentação e layout de conteúdo em páginas web, personalizando a aparência dos elementos HTML.

JavaScript

JavaScript é uma linguagem de programação amplamente usada para adicionar interatividade, comportamentos dinâmicos e manipulação de dados em páginas web e aplicativos, incluindo servidores.

GIT

Git é um sistema de controle de versão distribuído que rastreia alterações em arquivos e permite a colaboração em projetos de software.

GitHub

GitHub é uma plataforma de hospedagem de código-fonte, baseada em Git, que possibilita aos desenvolvedores armazenar, colaborar, revisar e gerenciar projetos de software de maneira pública ou privada.

Trello

Trello é uma ferramenta de gerenciamento de projetos visual baseada em quadros, listas e cartões, que permite a organização e colaboração de equipes, facilitando a aplicação de metodologias ágeis no desenvolvimento de projetos.

Skills em desenvolvimento

Typescript

TypeScript é uma linguagem de programação de código aberto que acrescenta tipagem estática opcional ao JavaScript, possibilitando uma detecção mais robusta de erros e simplificando o desenvolvimento de projetos complexos.

React

React é uma biblioteca JavaScript de código aberto do Facebook que possibilita a criação de interfaces de usuário interativas e dinâmicas para aplicativos web e móveis.

Angular

Angular é uma plataforma de desenvolvimento web de código aberto, mantida pelo Google, que permite a criação de aplicativos web modernos e dinâmicos, utilizando o padrão de arquitetura MVVM e a linguagem TypeScript.
